Person API This is a simple API to get statistics about your name Endpoints POST /api/people Request body example: { "firstName": "John", "lastName": "Doe" } firstName must be present. middleName and lastName are optional. GET /api/people Uses Request parameters to search for names. Examples: /api/people?firstName=John&lastName=Doe /api/people?firstName=John /api/people?name=John Doe Either firstName or name must be present. middleName and lastName are optional.
